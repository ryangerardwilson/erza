# erza

`erza` is an early-stage programming language project focused on building
command-line and terminal user interfaces.

## Direction

- Text-first language ergonomics
- Strong support for command dispatch and keyboard-driven flows
- Practical primitives for layout, state, styling, and event loops in the terminal
- A small, inspectable toolchain

## Status

This repository is intentionally starting small. The first phase is for naming,
scope, and language-design exploration before implementation hardens.
